# Romantic Landing Page

A new React + Vite project created in `romantic-site`.

## Features
- Luxury romantic storytelling experience
- Opening animation and love letter reveal
- Story timeline, gallery, reasons cards, bucket list, and memory constellation
- Final animated heart and personalized message
- Admin-friendly configuration via `src/config/experience.json`
- Dark/light theme toggle

## Run locally
```bash
cd romantic-site
npm install --legacy-peer-deps
npm run dev
```

## Build
```bash
npm run build
```

## Customize
Edit `src/config/experience.json` to update names, dates, text, gallery captions, story beats, reasons, and final message.
Add photo files to `public/photos` and reference them in the gallery entries.
